### Backend
Tecnologia: .NET 9.0
Porta: 5284
Variáveis de ambiente:
- DB_HOST (host)
- DB_PORT (porta)
- DB_NAME (nome do banco de dados)
- DB_USER (nome do usuario)
- DB_PASS (senha do usuario)

### Banco de dados
Tecnologia: MySQL
Porta: 3306
Variáveis de ambiente:
- MYSQL_ROOT_PASSWORD (senha raiz, pode ser "root")
- MYSQL_DATABASE (nome para uma database que será criada assim que o container for criado)
- MYSQL_USER (nome de um usuário que será criado assim que o container for criado)
- MYSQL_PASSWORD (senha do usuário que será criado assim que o container for criado)

### Frontend
Tecnologia: Node 20.0
Porta: 3000
Variáveis de ambiente:
- NEXT_PUBLIC_API_HOST (host da api que será usada)
- NEXT_PUBLIC_API_PORT (porta da api que será usada)
- NEXT_PUBLIC_API_ENP (endpoint da api que será usada, ex '/v1')
